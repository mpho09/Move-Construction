# Move-Construction
This is a modern construction company website built using HTML and CSS. This project showcases MOVE Construction’s Story of where they started, their services and contact information in a clean and user-friendly layout. This project shows front-end web development skills, including layout design, navigation and styling techniques.

Pages index.html — Home page with hero section, about summary, services overview and contact form. about.html — Dedicated About page with company background and mission. services.html — Full list of construction services offered. contact.html — Standalone contact page with form and company details.

Technologies Used HTML5 — Page structure and semantic markup. CSS3 — Styling, layout, colors, and basic responsiveness.

Functionality tested: All navigation links work and go to the correct pages. Contact form fields accept input correctly. All pages load without errors.

Browsers tested: Google Chrome

Screen sizes tested: Desktop (1024px+) Tablet (768px) Mobile (375px)

Resources Used MDN Web Docs. W3C HTML Validator. W3C CSS Validator. Google Fonts UIverse.io
